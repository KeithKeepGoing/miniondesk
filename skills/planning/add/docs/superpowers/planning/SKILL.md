## Skill: Planning

For complex tasks (>3 steps), create a plan BEFORE executing:

1. Read all relevant files to understand the current state
2. Break the task into atomic, independently-verifiable steps
3. Save the plan to `/workspace/group/docs/plans/{task-name}.md`
4. Execute one step at a time
5. After each step, verify it worked before proceeding

Plan format:
```
## Plan: {task name}

### Steps
- [ ] Step 1: Description
- [ ] Step 2: Description
...

### Verification
How to confirm the full task is complete
```

Update the plan file as you complete each step.
